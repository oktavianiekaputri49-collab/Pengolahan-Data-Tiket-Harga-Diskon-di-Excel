# Pengolahan-Data-Tiket-Harga-Diskon-di-Excel
Mini project menggunakan Microsoft Excel untuk mengolah data tiket penumpang berdasarkan kode tiket. Data diproses untuk menentukan kelas, jenis tiket, tahun lahir, harga awal, dan diskon. Project ini melatih pengolahan data terstruktur menggunakan fungsi Excel.
